# 🚀 Personal Portfolio Website

A modern, responsive, and animated portfolio website built to showcase my projects, skills, and experience.

---

## 🌐 Live Demo

👉 [https://portfolio.vercel.app](https://portfolio-website-six-psi-87.vercel.app/)

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🎬 Framer Motion
* 🌀 GSAP
* 🧠 TypeScript

---

## ✨ Features

* 🎯 Fully responsive (mobile + desktop)
* 🎬 Smooth animations and transitions
* 🧩 Component-based architecture
* ⚡ Fast performance with Vite
* 🌙 Modern dark UI
* 🖱️ Interactive hover effects
* 📜 Scroll-based animations

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/PrinceJoshi312/Portfolio-Website.git
cd Portfolio-Website
```

### 2. Install dependencies

```
npm install
```

### 3. Run development server

```
npm run dev
```

### 4. Build for production

```
npm run build
```

### 5. Preview production build

```
npm run preview
```

## 📦 Deployment
